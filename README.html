Premerger observation and characterization of super-massive black hole binaries#

This is the data release for the paper “Premerger observation and characterization of super-massive black hole binaries”, which can be found here. We release the data behind each of the figures and the code and configuration files that were used to produce each of them. We also demonstrate the steps needed to reproduce the search and parameter estimation using our code.

As always python, software dependencies will change in the coming years, and this code may be incompatible with future releases of any dependency. This paper was generated using a fork of PyCBC at icg-gravwaves/pycbc. This was forked from PyCBC at cd0e16a.

Reproducing the analysis environment#

We plan to have a docker image, but we also provide instructions here so that if the user wants to use our work as a basis for development, that is possible

We have provided a yaml file, which can be used to create a conda environment for the analysis:

conda env create -f install_reqs.yml

Make sure you have activated the environment using conda activate env_lisa_premerger

This will use the latest possible version of PyCBC, however we need some changes to to main code. So we will install a particular version of PyCBC. To get this, run

git clone git@github.com:icg-gravwaves/pycbc.git
cd pycbc
git branch lisa-pre-merger
pip install .
